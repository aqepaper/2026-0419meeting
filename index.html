原始規定條文
補充建議項目（橘底）
一
補課前
報到流程
提早10分鐘抵達 → 照客櫃台報到打勾 → 禮佛 → 美工室白板簽到 → 回補課教室
  • 提早十分鐘抵達，與前一班組員交接，護念彼此。
上線自我評量
  • 檢查平板有沒有電，發現不到50%電量的平板，放在充電層待充。（沒有使用的平板也會耗電）
  • 檢查抽屜內的轉接器和耳機是否相連未拔，如發現請分離歸在正確抽屜。
  • 檢查補課登記表是否已寫滿需繳回書記室。
  • 檢查已經過期的小平板設備，應盡快放回書記室門口櫃子下方塑膠籃。
  • 檢查清消物料是否不足，如果有缺，回報組長。
  • 跟上一班交接坡務內容後，也交接給比自己晚到的下一班師兄。
確認補課名單
  • 上線後主動至知客室確認當日預計來補課的學員名單與班別。
設備備妥檢查
  • 早班義工上線時先接上充電線，午班義工上線時取下充電線，將充好電的平板放在補課專車最上層（最上層只放置一台），其餘收在最下層。
  • 補課專車由早班義工從知客室櫃台內推出。
  • 逐一確認補課專車上各抽屜設備齊全：平板、耳機、轉接器、支架、借用登記表、補課登記表、護貝補課須知、筆筒。
  • 確認紅色資料夾內備用表格是否充足，發現剩一張時至知客室影印，至少留2張在資料夾內。
確認班表與交接
  • 上線時先觀察補課登記表和借用物品登記表，與現場實際使用狀況是否有出入，如有疑問趕快請教上一班的義工。
  • 交接內容涵蓋人、事、時、地、物，當有問題時多一句關心與確認姓名、法名，可以幫助每個環節釐清事實。
  • 交接時確認是否有逾期申請單夾在平板內尚未處理，若有，確認學員是否當日預計來補課。
  • 交接時確認是否有補課到一半尚未完成的學員，了解補到第幾分鐘，下次預約時段。
二
補課中
確認補課類型
  • 學員來時統一詢問：未超過一個月正常線上補課？逾期補課？申請重聽？補靜坐？
一般補課
  • 一般補課期限：缺課日隔週一起算，31日內完成。
  • 學員進入補課主頁後輸入姓名，若法名對不上，點「往下箭頭」找同名同姓的師兄。
  • 學員手機上線後查無補課內容，先判斷三種可能：已逾期、晚到早退出席表不算缺課、精舍系統尚未更新。
逾期補課
  • 先到知客室查詢紀錄表，確認缺哪一堂，再引導填寫逾期補課單。
  • 逾期補課單經法師簽核後，最快隔日設備才能準備好，當場確認學員下次來的日期與時段。
  • 學員逾期補課，有寫補課單但平板沒有檔案，依照學員班別取「班別USB」進行檔案拷貝至平板。
補課到一半中途離開
  • 在原登記單上寫補到第幾分鐘，修改下次日期與時段，單子與設備一同放回書記室，備註從第幾分鐘開始播放供下次接班義工參考。
  • 回報小組長，以利關心後續交接。
引導連線與操作硬體
  • 連線方式一：開相機掃描二維碼。
  • 連線方式二：Google輸入網址 zen.ctcm.org.tw。
第一次連線 開飛航 → 開WIFI → 連上 zenclass
有經驗學員 開飛航 → 開WIFI → 自動連上 zenclass
  • 未開飛航的隱憂：來電可能斷網、畫面當機、學員可能同時收發LINE，務必提醒學員先開飛航。
  • 幫學員設定手機的同時，提出補課須知護貝資料，引導學員閱讀注意事項。
座位引導
  • 男眾坐東單，女眾坐西單。
  • 空的長條桌，先引導學員坐靠兩側的位子，以免中央走道兩側位子入座時影響他人用功。
借用設備
  • 借用平板、耳機、支架、轉接器時，確實填寫該設備的號碼。
  • 由補課學員自行填寫借用登記表，義工於表格中點交簽名。
斷線處理
  • 回到補課系統首頁，重新登入一次，就會從補到一半的地方繼續播放。
  • 耐心向學員說明上列內容，使學員安心。
補課中注意事項
  • 學員開始補課後不打斷學員。
  • 19:30後來補課者，先請示知客室值班法師，詢問時表明學員班組別、姓名、法名及補課類型，由法師決定是否開方便。
  • 上午08:30-09:00和晚上18:30-19:00補課教室為學員長會議之用，義工暫時移動到大雄寶殿西單實木桌椅，引導來補課的學員。
  • 義工於值班期間機動檢查所有平板電量，發現不足50%應主動協助充電至100%。
三
補課後
確認補課完成
  • 逾期補課平板歸還時：申請單標示完成時間與簽名，連同平板拿到書記室門口書櫃下方盒子。
  • 補課到一半離開的學員：單子與設備放回書記室，備註從第幾分鐘開始播放。
  • 平板有夾逾期申請單但學員當日未來：當日最後一班將申請單和平板一併繳回書記室。
  • 補課登記表寫滿時：立刻送回書記室。
設備歸還點交
  • 義工於表格中點交簽名。
  • 逐一檢查每一個抽屜是否放置正確的設備，有缺少時立即回報組長。
  • 發現耳機與轉接線未分開，應分別放在正確的抽屜內。
  • 設備歸還時，取一顆酒精棉花球給學員自行清消。
引導學員離開
關飛航 → 關WIFI → 開行動數據 → 打開LINE測試
  • 感恩學員發心補課，提醒下次補課期限。
  • 學員未補完提前離開時，告知已記錄補到第幾分鐘，請學員確認下次預約時段。

設備損壞處理
義工發現 → 回報小組長 → 小組長測試 → 確認報修
  • 發現損壞設備立即標示，不放回原位繼續流通。
  • 報修後追蹤維修進度，回報組長確認修復完成。
冷氣使用規範
  • 一般冷氣僅開後區，補課人數（不含義工）達4人時可增開中區，前區不使用。
  • 晚班皆無學員補課，可視情況提早關閉全部冷氣，開後門通風。
  • 開關冷氣權責：由當班義工判斷，不確定時請示組長。
五
交接班
交接程序
  • 提早十分鐘抵達，與前一班組員交接，護念彼此。
  • 上線時先觀察補課登記表和借用物品登記表，與現場實際使用狀況是否有出入，如有疑問趕快請教上一班的義工。
  • 交接內容涵蓋人、事、時、地、物，當有問題時多一句關心與確認姓名、法名，可以幫助每個環節釐清事實。
  • 跟上一班交接坡務內容後，也交接給比自己晚到的下一班師兄。
  • 交接時口頭確認當班是否有異常狀況、待處理事項、需回報組長的事件。
物料點交
  • 逐一檢查每一個抽屜是否放置正確的設備，有缺少時立即回報組長。
  • 發現耳機與轉接線未分開，應分別放在正確的抽屜內。
  • 檢查平板電量，發現不到50%電量的平板，放在充電層待充。
  • 檢查抽屜內的轉接器和耳機是否相連未拔，如發現請分離歸在正確抽屜。
  • 檢查清消物料是否不足，如果有缺，回報組長。
  • 檢查已經過期的小平板設備，應盡快放回書記室門口櫃子下方塑膠籃。
  • 確認紅色資料夾內備用表格是否充足。
  • 確認補課專車上筆筒內的筆是否有水，發現沒水的筆直接丟棄。
登記表確認
  • 檢查補課登記表是否已寫滿需繳回書記室。
  • 確認借用物品登記表與現場實際設備數量是否相符。
  • 確認是否有逾期申請單夾在平板內尚未處理。
  • 確認是否有補課到一半尚未完成的學員紀錄，了解補到第幾分鐘及下次預約時段。
與照客交接
  • 補課設備登記表交接給照客後，請照客在單子上簽名確認點交完成。
  • 補課專車推回知客室時，物料擺放整齊在正確的架子上，維護櫃台門面整潔。
  • 若精舍未關山門且下一班無義工，柔軟語請示照客組長幫忙關心可能即將來的補課學員。
  • 照客未簽名時，主動再次請照客確認並簽名，不帶走未完成點交的設備。
簽退
  • 於美工室白板完成簽退時間。
  • 簽退前確認個人物品已帶走，勿遺留於補課教室。
六
接待禮儀
迎接學員
  • 義工出坡時穿著橘T，胸前配戴名牌。
  • 學員進入補課教室時，主動起身迎接，確認來意。
  • 學員來時統一詢問：未超過一個月正常線上補課？逾期補課？申請重聽？補靜坐？
  • 以柔軟語引導學員，態度親切，使學員安心。
引導就座
  • 男眾坐東單，女眾坐西單。
  • 空的長條桌，先引導學員坐靠兩側的位子，以免中央走道兩側位子入座時影響他人用功。
  • 引導就座後，協助學員設定手機連線，同時提出補課須知護貝資料引導閱讀。
居士服勸導
  • 學員未穿居士服時，口頭柔軟勸導：「補課視同正式上課，請穿著居士服。初級班學員可著便服。」
  • 詢問學員：「請問您今天有帶居士服嗎？」並協助填寫未著居士服登記表。
  • 如需購買居士服，告知可請學長協助至中台世界博物館禮品部購買。
補課中關懷
  • 網路斷線時，耐心向學員說明回到補課系統首頁重新登入即可從補到一半的地方繼續，使學員安心。
  • 學員補課時做出違規舉動，不直接碰觸學員身體，以柔軟語提醒。
  • 學員開始補課後不打斷學員。
  • 請勿在補課區後方聊天交談講話，以免影響補課師兄用功。
  • 義工保持用功狀態，以身作則護念補課學員。
特殊狀況應對
  • 19:30後來補課者，先請示知客室值班法師，詢問時表明學員班組別、姓名、法名及補課類型，由法師決定是否開方便。
  • 上午08:30-09:00和晚上18:30-19:00補課教室為學員長會議之用，義工暫時移動到大雄寶殿西單實木桌椅，引導來補課的學員。
  • 夜研學員週一18:30來補課，若影片內容會超過19:30，柔軟語告知師兄19:30為正式課程，把握時間進禪堂用功為主；並於19:20提醒學員上課時間快到了，沒有補完的範圍擇日再補。
  • 無法解決問題時，覺察自己的煩躁，立刻停止惡念，先安撫學員，勇於請教組長或照客。
  • 遇到設備問題無法解決時，柔軟語告知學員：「自己正在學習中，很抱歉耽誤您寶貴的時間，趕快再幫忙詢問看看。」
學員離開
  • 引導學員離開補課教室前恢復手機正常狀態：關飛航 → 關WIFI → 開行動數據 → 打開LINE測試可否正常收發訊息。
  • 感恩學員發心補課，提醒下次補課期限。
  • 學員未補完提前離開時，告知已記錄補到第幾分鐘，請學員確認下次預約時段。
電話應對
注意：以下為建議建立之SOP，附件資料無相關內容。
  • 接聽電話開場白：「阿彌陀佛，這裡是補課組，請問有什麼需要協助的？」
  • 學員來電詢問補課期限：告知缺課日隔週一起算，31日內完成。
  • 學員來電詢問如何連線：告知開飛航→開WIFI→連上zenclass，或輸入網址zen.ctcm.org.tw。
  • 學員來電詢問逾期如何處理：告知需來精舍填寫逾期補課單，經法師簽核後隔日安排設備。
  • 無法回答的問題：告知學員稍後由組長回覆，並立即回報組長處理。
七
請假程序
請假方式
  • 提前告知組長，說明請假日期與時段。
  • 如果當天臨時請假，請組長協助告知交接班的師兄，以免夥伴空等。
  • 除了告知組長以外，也同時告知同一班的組員自己會晚到或請假。
請假類別與補班規則
  • 請假須補班、換班、或代班。
  • 代班是福報存款，不當作補班。
  • 請假可以向師兄換班，換班之後不用再補班。
  • 預約看診需要補班。
出勤異常處理
  • 義工無故消失，無法聯繫，也沒有回報請假，也沒有讀LINE訊息，3天後進行通報。
八
一般守則
服裝
  • 補課組義工出坡時穿著橘T，胸前配戴名牌。
  • 不建議穿居士服，容易跟補課學員混淆辨識度。
物料管理
  • 清消物料不足時，每一班隨時關心不足情況，回報任何一位組長，組長回報執秘安排採購。
  • 不織布剪成8小張放在鐵盒子裡，義工下線前取用小張的清消桌椅。
  • 紅色資料夾內，如發現剩一張，請拿到知客室影印，並至少留2張在資料夾內。
  • 如發現筆沒水，請直接丟掉，勿再放回筆筒。
個人物品
  • 義工個人物品請勿放置於補課教室，下哨前確認個人物品已帶走。
用功守則
  • 請勿在補課區後方聊天交談講話，以免影響補課師兄用功。
  • 補課操作系統還不熟悉時，可預約組長在非值班的時間來精舍學習，反覆練習。
  • 每次出坡時向已經學會的組員請教，請教時可在大雄寶殿西單桌椅區討論。
財產與設備
  • 平板或耳機等設備損壞，義工回報小組長，小組長測試，若有問題就報修。
  • 小平板補課設備如果沒有電，要繳回書記室，因為補課組沒有對應的充電線。
  • 財產清冊製作中，各班義工協助確認設備數量與狀況。
